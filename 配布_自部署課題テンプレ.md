## 6. やってほしいこと（🔒 Claude Code への指示）

1. 上の要件（1〜5）で、**動く最小モック**を作る（まずは入力→処理→出力が1本通ればOK）。
2. 起動手順を **README** にまとめる。

4. あとから直しやすいよう、**変えたくなる場所（プロンプト・パラメータ）を分かりやすく**しておく。
