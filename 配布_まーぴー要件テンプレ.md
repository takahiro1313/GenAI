> このmdを **Claude Code にそのまま渡す** と、音声で話せる「まーぴー」を作り始めます。
> **🟢 ＝あなたが決めるところ**（`___` を埋める／候補から選ぶ）。**🔒 ＝触らないところ**（骨組み。そのままでOK）。
> まずは 🟢 を埋めて、Claude Code に「この要件でまーぴーを作って」と伝えてください。

---

## 1. 作るもの（🔒 そのまま）

ブラウザのマイクに話しかけると、**音声でリアルタイムに応答するカスタマーサポートbot「まーぴー」**（青い小鳥）。
まーぴーの画像が表示され、**声に合わせて動く**。会話は文字でも表示される。

- スタック（🔒）：Streamlit ＋ OpenAI Realtime API（`gpt-realtime`）＋ WebRTC ＋ Web Audio。
- セキュリティ（🔒）：APIキーは `.env`。ブラウザには**短命トークンのみ**渡す（永続キーを出さない）。

## 5. やってほしいこと（🔒 Claude Code への指示）

1. 上の要件で、音声で話せるまーぴーを **Streamlit アプリ** として作る。
2. 起動手順（`pip install` → `.env` 設定 → `streamlit run` → マイク許可 → 話す）を **README** にまとめる。
3. **口調を変えたい時に直す場所（プロンプト1箇所）** を分かりやすくしておく。
4. APIキーは `.env`、ブラウザには短命トークンのみ。`.env.example` も用意する。
